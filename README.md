# BiblioSystem

## Descripción
Sistema de gestión bibliotecaria con interfaz gráfica.

## Cómo ejecutar
1. Abrir NetBeans
2. Ejecutar main/BiblioSystem.java

## Credenciales
- Admin: admin / admin
- Operador: crear desde admin
- Estudiante: auto-registro

## Funcionalidades
- Gestión de libros 
- Préstamos y devoluciones
- Historial de préstamos
- Reportes HTML
- Bitácora de operaciones

## Problemas encontrados
- Los libros no se guardan entre sesiones (Así era, creo)
- Al registrar estudiante, la contraseña se guarda en texto plano (sin encriptación)
- No hay opción para recuperar contraseña olvidada

## Mejoras posibles
- Implementar encriptación de contraseñas
- Agregar opción para modificar datos del estudiante
- Permitir exportar reportes en otros formatos como PDF
- Agregar búsqueda avanzada de libros por género o año
